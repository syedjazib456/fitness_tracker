# 🏋️ Fitness Tracker – MERN Stack Web Application

A full-stack Fitness Tracker application built using the MERN stack (MongoDB, Express.js, React.js, Node.js) that helps users track workouts, monitor progress, and maintain a healthy lifestyle.

---

## 🚀 Features

- 🔐 User Authentication (JWT Login/Register)
- 🏃 Add, Update, Delete Workout Sessions
- 📊 Track Daily/Weekly Fitness Progress
- 📅 Workout History Management
- 🎯 Set Fitness Goals
- 📈 Progress Dashboard with Charts
- 👤 User Profile Management
- 📱 Fully Responsive UI (Mobile + Desktop)

---

## 🧰 Tech Stack

**Frontend:**
- React.js
- Redux / Context API
- Bootstrap / Tailwind CSS
- Chart.js (for analytics)

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (Mongoose ODM)

**Authentication:**
- JSON Web Token (JWT)

---

## 📂 Project Structure
/client → React Frontend
/server → Node.js Backend

## ⚙️ Installation & Setup

## 1️⃣ Clone the repository

git clone https://github.com/yourusername/fitness-tracker.git

cd server
npm install

cd client
npm install


MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000

## **Start Backend**
cd server
npm start

## **Start Frontend**
cd client
npm start
